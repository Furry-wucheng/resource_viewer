# Design — 页面原型

> 本目录存放各个页面的高保真原型图，格式为 HTML。

## 约定

1. 每个页面一个独立的 HTML 文件
2. 文件名使用英文小写 + 短横线，见名知意
3. 页面原型包含真实的布局和交互示意，但不包含实际数据加载逻辑
4. 在 PRD 文档中通过 `@design/xxx.html` 方式引用
