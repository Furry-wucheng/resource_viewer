# Design — 页面原型

> 本目录存放各个页面的基本原型图，格式为 HTML。
> 已完成：v1.2，2026-06-19
> 每个页面均已内联 JS 实现关键交互演示，零外部依赖。交互通过页面原生 UI 驱动。

## 约定

1. 每个页面一个独立的 HTML 文件
2. 文件名使用英文小写 + 短横线，见名知意
3. 页面原型包含布局和可交互的演示
4. 在 PRD 文档中通过 `@design/xxx.html` 方式引用
5. 公共样式抽取到 `design-tokens.css`，所有页面共享 CSS 变量
6. JS 内联在 `<script>` 中，零外部依赖

## 查看器设计要点

- **内容填满全屏**：图片/视频始终撑满视口，控件悬浮叠加不挤占内容空间
- **无极显翻页箭头**：翻页通过左右半屏点击 + 滑动实现
- **底部一行布局**：图片模式 = 页码 + 滑动条 + 全屏按钮；视频模式 = 已播时间 + 进度条 + 总时长 + 暂停 + 全屏按钮
- **双页模式**：宽度 ≥ 900dp 可用，两张图并排填满；不足时自动禁用
- **移动端**：仅查看器支持自动旋转（含视频全屏按钮触发），其余页面锁定竖屏

## 设计令牌 (`design-tokens.css`)

- CSS 变量统一管理颜色、间距、字体、圆角、阴影
- 支持浅色/深色主题切换（通过 `data-theme` 属性）
- 标签 12 色预设
- 响应式网格自适应列数：3→4→5→6
- 宽屏模式（≥900dp）侧栏双栏布局
