## 2.9 错误处理基础设施

### 执行目标
按 `@tech/05-错误处理策略.md` 实现 DomainError 类型树、Result\<T\>、UiState、Progress\<T\>、BaseViewModel。

### 任务边界
- 纯 Dart 基础设施，不依赖 Flutter
- Repository 错误目前抛出原始异常（阶段 03+ 包装为 Result）

### 产出物
| 文件 | 说明 |
|------|------|
| `lib/domain/models/domain_error.dart` | `DomainError` sealed class + 12 个子类（均加 `final` 修饰） |
| `lib/domain/models/result.dart` | `Result<T>` sealed class（`Ok` / `Err`） |
| `lib/domain/models/progress.dart` | `Progress<T>` sealed class（`ProgressUpdate` / `ProgressDone` / `ProgressError`） |
| `lib/ui/core/view_models/base_view_model.dart` | `BaseViewModel` 抽象类（state/errorMessage/lastError/canRetry） |

### 验收标准
- [ ] `DomainError` sealed class 含 12 个子类型（均加 `final` 修饰）：`SourceUnreachableError`, `SourceAuthError`, `FileNotFoundError`, `FileAccessDeniedError`, `NetworkTimeoutError`, `MediaLoadError`(含 `MediaType` 枚举), `MediaEncryptedError`(含 `mediaType`), `ValidationError`, `OperationCancelledError`, `InsufficientStorageError`, `UnsupportedFormatError`(含 `format`), `DatabaseError`
- [ ] `MediaType` 枚举：`image` / `pdf` / `archive` / `video`（与 `ResourceType` 解耦）
- [ ] `Result<T>` 含 `Ok(value)` 和 `Err(error)`
- [ ] `Progress<T>` 含 `ProgressUpdate(value, current, total)`, `ProgressDone(result)`, `ProgressError(error)`
- [ ] `BaseViewModel` 含 `state` (UiState), `errorMessage`, `lastError`, `canRetry`, `_startLoading()`, `_setResult()`, `retry()`
- [ ] `canRetry` 逻辑：`SourceUnreachableError`/`FileNotFoundError`/`NetworkTimeoutError`/`MediaLoadError` 返回 true，其余 false
- [ ] `_mapError` 中文文案映射完整
- [ ] 单元测试：每种 Error 类型的 canRetry 判定，Result 的 switch 穷举
