# 阶段 03 — 本地源链路

> **定位**：Core | **子阶段数**：20 | **前置依赖**：01（脚手架）, 02（数据层）
> **目标**：首个端到端用户链路 — 添加本地文件夹 → 扫描入库 → 首页缩略图网格 → 查看器翻页/视频播放

---

## 阶段总体目标

交付从"添加本地文件夹数据源 → 文件浏览器勾选 → 扫描为 Resource → 首页缩略图网格展示 → 点击进入查看器翻页/播放视频"的完整闭环。此阶段完成后，用户可用本地文件夹进行完整的收藏-浏览体验。

### 验收标准
- [ ] 响应式列数：<600dp 三列 / 600~900 四列 / 900~1200 五列 / >1200 六列
- [ ] GridView.builder + `SliverGridDelegateWithMaxCrossAxisExtent(maxCrossAxisExtent: 180)`
- [ ] `childAspectRatio: 2/3`
- [ ] `cacheExtent: 1500`
- [ ] 缩略图：`Image.file(cacheWidth: 180, cacheHeight: 270, fit: BoxFit.cover)`
- [ ] 文件名最多 2 行，溢出省略号，12sp
- [ ] PDF 和压缩包显示类型角标（右下角图标）
- [ ] 文件夹无角标
- [ ] 单击资源 → 跳转查看器 `/viewer/:resourceId`

- [ ] 加载状态：顶部扫描进度条，已扫描数量实时更新
- [ ] 缩略图加载失败 → 显示类型占位图
- [ ] Tab 切回时保持滚动位置（`PageStorageKey`）
- [ ] 与 `@design/homepage.html` 布局一致
