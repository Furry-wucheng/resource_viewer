## 7.5 ArchiveProvider 实现（ZIP 压缩包阅读）

### 执行目标
使用 `archive` 包实现 ZIP 压缩包的内存阅读，不解压到磁盘。

### 任务边界
- 仅支持 ZIP（RAR/7Z 为 P2）
- 密码保护的 ZIP 弹窗输入密码

### 产出物
| 文件 | 说明 |
|------|------|
| `lib/shared/content_provider/archive_provider.dart` | `ArchiveProvider` |
| `lib/data/services/archive_service.dart` | `ArchiveService`（archive 包封装） |

### 验收标准
- [ ] `pageCount` = ZIP 内图片文件数量（按文件名排序）
- [ ] `loadPage(index)` → 内存解压对应图片，返回 `Uint8List`
- [ ] 仅展示图片文件（jpg/png/gif/webp），跳过其他文件
- [ ] 密码保护的 ZIP → 弹窗输入密码 → 验证 → 继续或提示错误
- [ ] `dispose()` → 释放内存中解压的数据
- [ ] 单元测试：创建测试 ZIP → 验证页码 + 内容读取
