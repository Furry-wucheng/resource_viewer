# 02 — 标签系统

> 所属模块：B · 优先级：P0 · 版本：v1.2 · 日期：2026-06-17 · 状态：已定稿

---

## 1. 模块概述

标签系统是 Resource Viewer 的核心组织能力。标签分为两类：

- **内置标签**：系统预置，不可删除、不可重命名。当前仅包含"收藏"标签（⭐）。
- **自定义标签**：用户自由创建，支持颜色标记、重命名、删除。

用户可为一个资源绑定多个标签，并在首页通过标签组合筛选快速定位目标资源。

与文件夹层级不同，标签提供**扁平、多维、灵活**的分类体系——同一部漫画可以同时是"热血""已完结""汉化"，外加一个"收藏"标记。

## 4. 详细需求

### 4.1 内置标签

系统在首次启动时自动创建内置标签，内置标签具有以下特殊属性：

| 属性 | 说明 |
|------|------|
| `isBuiltIn = true` | 数据层标识，不可移除 |
| 不可删除 | 删除按钮隐藏，即使用户尝试删除也拒绝 |
| 不可重命名 | 名称锁定 |
| 颜色固定 | 收藏标签颜色固定为星标金色 `#FFC107` |
| 自动创建 | 应用首次启动时自动创建，若不存在则补建 |

**当前内置标签**：

| 名称 | 颜色 | 用途 |
|------|------|------|
| 收藏 | `#FFC107`（金色） | 标记用户常用的/喜欢的资源 |

> 未来可能扩展其他内置标签，如"最近阅读"等，但当前仅此一个。

### 4.2 自定义标签创建

**入口**：标签管理页的"+"按钮 / 资源详情编辑标签时可快速创建

**创建表单**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 标签名称 | 文本 | 是 | 1-20 个字符，不可为空，不可与内置标签同名 |
| 标签颜色 | 选择 | 否 | 从预设 12 色中选择，默认取第一个未使用的颜色 |

**预设颜色**（12 色）：

```
红 #E53935    粉 #D81B60    紫 #8E24AA    深蓝 #1E88E5
蓝 #039BE5    青 #00ACC1    绿 #43A047    黄绿 #7CB342
橙 #FB8C00    棕 #6D4C41    蓝灰 #546E7A   灰 #757575
```

**约束**：
- 标签名称唯一，重复时提示"标签名已存在"
- 不可与内置标签同名（如不可创建名为"收藏"的标签）
- 创建成功后自动返回上一页或更新列表

### 4.3 标签编辑 & 删除

**内置标签**：不显示重命名/删除/修改颜色选项。

**自定义标签**：

**重命名**：
- 长按标签项（移动端）/ 右键（桌面端）→ 弹出菜单 → 选"重命名"
- 行内编辑或弹窗编辑，确认后更新标签名
- 唯一性校验同上

**修改颜色**：
- 同样从弹出菜单中选择"修改颜色"→ 显示 12 色选择器 → 点选确认

**删除**：
- 弹出菜单 → "删除"→ 二次确认弹窗："删除标签'xxx'将解除所有资源的关联，确定吗？"
- 确认后删除标签及所有 ResourceTag 关联记录

### 4.4 标签与资源关联

**入口**：资源详情弹窗 / 资源查看器工具栏 → "编辑标签"

**交互**：
- 以多选列表展示全部标签：内置标签在最顶部，自定义标签在下方
- 内置"收藏"标签用星标图标展示，点击切换收藏/取消（与其他标签操作一致）
- 已关联的标签显示勾选状态
- 点击切换勾选/取消
- 列表底部有"新建标签"入口，可即时创建并自动勾选
- 确认后保存变更

**快捷收藏**：
- 首页缩略图右下角显示星标图标，点击直接切换收藏状态（无需进入详情）
- 查看器工具栏有星标按钮，同样直接切换

**约束**：
- 一个资源可关联 0-N 个标签
- 一个标签可关联 0-N 个资源

### 4.5 标签筛选

**位置**：首页顶部，资源网格上方

**交互规则**：

```
┌──────────────────────────────────────────────┐
│ [🔲 全部] [⭐ 收藏] [热血] [已完结] [汉化] ...│  ← 横向滚动
└──────────────────────────────────────────────┘
 |         |          └─────────────────────┘
 |         |               自定义标签（可滚动）
 |         └─ 内置：收藏
 └─ 固定：全部
```

| 操作 | 行为 |
|------|------|
| 点击"全部" | 取消所有筛选（包括收藏），显示全部资源 |
| 点击"收藏" | 仅显示拥有内置"收藏"标签的资源 |
| 同时选中"收藏" + 自定义标签 | 取交集：收藏 且 拥有该标签的资源 |
| 点击未选中自定义标签 | 选中该标签，取与当前筛选的交集 |
| 点击已选中自定义标签 | 取消选中，列表更新 |
| 选中多个自定义标签 | 取**交集**（同时拥有所有选中标签的资源） |
| 未选中任何标签（含全部和收藏） | 显示全部资源 |

**视觉反馈**：
- 选中的筛选项高亮显示："全部"和"收藏"用实心背景，"收藏"用金色星标 + 背景
- 自定义标签选中时填充标签颜色背景
- 未选中的灰色描边样式
- "全部"和"收藏"固定在左侧，不随滚动隐藏

**筛选结果提示**：
- 筛选后显示提示文本："筛选出 12 / 56 个资源"
- 筛选结果为空时显示："没有匹配的资源，试试调整筛选条件"

**超出屏幕处理**：
- 自定义标签栏支持横向滚动
- "全部"和"收藏"固定在最左侧，不随滚动隐藏

### 4.6 标签管理页

**页面结构**：

```
┌─────────────────────────────────────┐
│  ← 标签管理                    [+]  │  ← 顶部导航栏
├─────────────────────────────────────┤
│  🔍 搜索标签...                     │  ← 搜索框（P2）
├─────────────────────────────────────┤
│  ⭐ 收藏 (内置)       共 12 个资源   │  ← 内置标签，带锁定标识，无操作菜单
├─────────────────────────────────────┤
│  ● 热血              共 12 个资源   │  ← 自定义标签项
│  ● 已完结             共  8 个资源   │  ← 显示关联资源数
│  ● 汉化               共 15 个资源   │
│  ● 未读               共  3 个资源   │
│  ...                                │
└─────────────────────────────────────┘
```

- 内置标签固定显示在列表最顶部，与其他标签用分割线隔开
- 内置标签显示 `(内置)` 标记或锁定图标，不可编辑/删除
- 自定义标签：颜色圆点 + 标签名 + 关联资源数量
- 点击标签项 → 跳转首页并自动预置该标签筛选条件
- 长按/右键自定义标签 → 弹出操作菜单（重命名/修改颜色/删除）
- 列表排列：内置标签在前 → 自定义标签按创建时间倒序

### 4.7 文件浏览器中的标签操作

标签操作在文件浏览器中有以下入口：

#### 4.7.1 单独打标签（正常模式）

- 长按/右键已入库的文件夹或文件 → 弹出菜单 → "打标签"
- 弹出标签多选弹窗，勾选/取消即可
- 未入库的项不显示"打标签"选项，仅显示"添加到资源库"

#### 4.7.2 批量打标签（多选模式）

文件浏览器多选 N 项 → 底部"批量打标签"→ 流程同 `@03-数据源管理.md` 4.6：

```
① 过滤：遍历勾选项，检查 sourceId + relativePath 是否存在 Resource 记录
     → 已入库：纳入打标签范围
     → 未入库：跳过

② 弹窗提示："将为 5 个资源打标签（已跳过 2 个未入库项）"

③ 标签多选弹窗 → 勾选 → 确认

④ 批量写入 ResourceTag 记录
```

#### 4.7.3 添加资源时附带打标签

文件浏览器批量添加资源时，在确认流程中可附加打标签（见 `@03-数据源管理.md` 4.5.1 第③步）。

#### 4.7.4 扫描入库 / 拆分资源时附带打标签

通过 ResourcePicker 弹窗扫描入库或拆分资源后，在确认流程中可附加打标签：

- ResourcePicker 确认选择 → 弹出标签选择弹窗（复用标签多选组件）
- 标签**完全重新选择**，不继承原资源标签
- 选择"跳过"则仅添加/拆分资源，不绑定标签

## 5. 边界条件

1. **标签数量为 0**（仅剩内置标签）：标签管理页显示空状态引导"创建第一个标签吧"
2. **标签名为空或全空格**：保存按钮禁用，输入框下方红色提示"标签名不能为空"
3. **标签名与内置标签同名**：弹 toast "'收藏'是系统内置标签，请换一个名称"
4. **删除标签时确认**：二次确认弹窗，防止误删
5. **筛选结果为空**：显示专属空状态插图 + "没有匹配的资源" + "清除筛选"按钮
6. **标签颜色用完**：12 色全部被占用后，新建标签默认使用灰色
7. **标签名重复**：保存时校验，弹 toast "标签'xxx'已存在"
8. **内置标签被意外删除**（异常情况）：应用启动时检测并自动补建
