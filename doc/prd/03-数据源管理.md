# 03 — 数据源管理

> 所属模块：C · 优先级：P0 · 版本：v1.2 · 日期：2026-06-17 · 状态：已定稿

---

## 1. 模块概述

数据源管理是资源的"入口"。用户在此添加本地文件夹、SMB 网络共享等数据来源，通过内置的文件浏览器手动勾选需要纳入资源库的文件夹或文件，触发扫描后这些资源即出现在首页。

---

## 2. 用户故事

| ID | 角色 | 场景 | 期望结果 |
|----|------|------|----------|
| US-C01 | 收藏用户 | 漫画存在本地 D 盘，想把它作为数据源 | 通过文件夹选择器添加路径，连接成功后源列表新增一项 |
| US-C02 | 收藏用户 | NAS 上有大量漫画，想通过 SMB 连接 | 填写 SMB 地址和凭据，测试成功后添加为源 |
| US-C03 | 收藏用户 | 添加源后不想全量导入，只想挑其中几个漫画文件夹 | 在文件浏览器中手动勾选需要的子文件夹，批量添加为资源 |
| US-C04 | 收藏用户 | 想看看 SMB 源下有哪些漫画文件夹 | 文件浏览器按目录树展示源内文件结构，支持列表和网格两种视图 |
| US-C05 | 收藏用户 | NAS 断网了，但本地源还能正常使用 | SMB 源显示"不可用"标记，本地源不受影响 |
| US-C06 | 收藏用户 | 某个源不再需要了，想移除 | 手动确认后删除源及关联资源与标签关联，标签本身保留 |
| US-C07 | 收藏用户 | 源内新增了一些漫画，想刷新看到新内容 | 手动触发同步，扫描新增/变更的文件 |
| US-C08 | 收藏用户 | 想把某个文件夹及其所有子文件夹一次性添加为资源 | 长按文件夹 → 扫描入库 → ResourcePicker 弹窗树形勾选全部子文件夹，批量添加 |
| US-C09 | 收藏用户 | 添加资源的同时想顺便打上标签 | 批量添加确认前弹出标签选择弹窗，可选的附加打标签 |
| US-C10 | 收藏用户 | 想给已入库的文件夹快速打标签（含其子文件夹） | ResourcePicker + 批量打标签；或长按/右键单个已入库项 → 打标签 |

## 4. 详细需求

### 4.1 添加数据源

#### 4.1.1 入口 & 流程

```
数据源管理页 → 点击 [+] 按钮 → 弹出选择菜单
  ├── "添加本地文件夹" → 系统文件夹选择器 → 选择路径 → 添加成功
  └── "添加 SMB 网络共享" → SMB 配置表单 → 填写 → 测试连接 → 添加成功
```

#### 4.1.2 本地文件夹源

- 调用系统原生文件夹选择器
- 选择路径后自动填充源名称（取文件夹名）
- 源名称可编辑
- 添加成功后自动打开该源的文件浏览器

#### 4.1.3 SMB 网络源

**配置表单**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 源名称 | 文本 | 是 | 用户自定义显示名称 |
| SMB 地址 | 文本 | 是 | 格式：`\\192.168.1.100\share` 或 `smb://host/share` |
| 端口 | 数字 | 否 | 默认 445 |
| 用户名 | 文本 | 否 | 匿名访问可为空 |
| 密码 | 密码 | 否 | 匿名访问可为空 |
| 域 | 文本 | 否 | 工作组/域名 |

**测试连接**：
- 填写完成后点击"测试连接"按钮
- 成功：绿色勾 + "连接成功"
- 失败：红色叉 + 具体错误原因（超时/认证失败/路径不存在）
- 测试成功后"添加"按钮才可点击

**凭据安全**：密码字段需安全存储，界面展示时脱敏显示 `****`

### 4.2 源列表

**列表项布局**：

```
┌──────────────────────────────────────────┐
│ [图标] 源名称            [开关] [···]    │
│        类型 | 12 个资源                   │
└──────────────────────────────────────────┘
```

- 图标区分源类型（本地 / SMB / FTP 等）
- 开关：关闭后该源资源不在首页显示
- `···` 菜单：重命名 / **编辑 SMB 凭据**（仅 SMB 源）/ 手动同步 / 删除

**状态指示**：

| 状态 | 视觉表现 |
|------|---------|
| 正常 | 正常显示 |
| 不可达（SMB 断网/路径不存在） | 置灰 + 警告图标 + "不可用"文字 |
| 扫描中 | 加载动画 + "扫描中..." |

### 4.3 文件浏览器

**入口**：点击源列表项 → 进入该源的文件浏览器

#### 4.3.1 布局模式

文件浏览器根据屏幕宽度自适应布局。

**窄屏（< 900dp）**：列表/网格 全屏切换

```
列表视图                           网格视图
┌──────────────────────────┐    ┌──────────────────────────┐
│ ← 返回  漫画源  [≡][🔲]🔤│    │ ← 返回  漫画源  [≡][🔲]🔤│
├──────────────────────────┤    ├──────────────────────────┤
│ 📁 > Comics > 海贼王      │    │ 📁 > Comics > 海贼王      │
├──────────────────────────┤    ├──────────────────────────┤
│ 📁 卷01_东海篇            │    │ ┌──┐ ┌──┐ ┌──┐ ┌──┐  │
│ 📄 设定集.pdf             │    │ │封│ │封│ │封│ │封│  │
│ 🎬 OP_特典.mp4           │    │ │面│ │面│ │面│ │面│  │
└──────────────────────────┘    │ └──┘ └──┘ └──┘ └──┘  │
                                │ 卷01  卷02  卷03  卷04  │
                                └──────────────────────────┘
```

**宽屏（≥ 900dp）**：左侧目录树 + 右侧内容区（列表/网格）

```
┌─────────────────┬──────────────────────────────────┐
│ ← 返回  漫画源   │  📁 > Comics > 海贼王     [🔲]🔤│  ← 右侧顶部
├─────────────────┼──────────────────────────────────┤
│ 📁 Comics       │  ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐    │
│  📁 海贼王       │  │封│ │封│ │封│ │封│ │封│    │
│    📁 卷01      │  │面│ │面│ │面│ │面│ │面│    │
│    📁 卷02      │  └──┘ └──┘ └──┘ └──┘ └──┘    │
│    📁 卷03      │  卷01  卷02  卷03 设定  OP     │  ← 右侧网格
│  📁 进击的巨人   │                                  │
│  📁 鬼灭之刃    │  (或列表模式)                     │
│  📄 设定集.pdf  │                                  │
├─────────────────┴──────────────────────────────────┤
│ 底部操作栏（正常模式无）                               │
└────────────────────────────────────────────────────┘
```

- 左侧目录树：**纯导航**，点击节点→右侧内容区更新；支持展开/折叠
- 右侧内容区：复用窄屏的列表/网格视图，独立排序
- 左侧树当前选中节点高亮
- 切换按钮：`[≡]` 列表 / `[🔲]` 网格，仅控制右侧内容区

**网格视图缩略图规则**（目的：快速预览文件内容，与资源库首页缩略图目的不同）：

| 项类型 | 网格缩略图 |
|--------|-----------|
| 文件夹（下层含图片/视频文件） | 取子文件夹内第一张图作为封面 |
| 文件夹（下层无图片/视频） | 显示文件夹占位图标 |
| 图片文件 | 自身缩略图 |
| 视频文件 | 首帧或视频占位图标 |
| PDF | 第一页渲染 |

#### 4.3.2 正常模式交互

| 操作 | 行为 |
|------|------|
| 点击文件夹 | 进入下一级目录 |
| 返回键 / 顶部←按钮 | 返回上一级目录（每按一次回退一级） |
| 面包屑导航 | 点击任意层级跨级跳转 |
| 点击图片/PDF/GIF/视频 | 进入统一查看器，可左右滑动浏览当前目录下所有兼容文件（视频与图片混合浏览，见 `@04-资源查看器.md` §4.4） |
| 长按/右键已入库项 | 弹出菜单：打开 / **打标签** / 属性 |
| 长按/右键未入库项 | 弹出菜单：打开 / **扫描入库** / 添加到资源库 / 属性 |

**长按/右键菜单判断**：

```
已入库的文件夹/文件：
  ├── 打开
  ├── 打标签               ← 进入标签多选弹窗
  └── 属性

未入库的文件夹/文件：
  ├── 打开
  ├── 扫描入库              ← 打开 ResourcePicker 弹窗（树形扫描选择，见 @06-页面结构与交互.md）
  ├── 添加到资源库          ← 直接添加，触发扫描，可选打标签
  └── 属性
```

#### 4.3.3 文件浏览器内查看

在文件浏览器中点击图片/PDF/GIF 等兼容文件进入查看器。**注意**：这与从资源库首页进入查看器的行为不同，详见下表：

| 维度 | 文件浏览器进入 | 资源库首页进入 |
|------|--------------|--------------|
| 翻页范围 | 当前目录下同层级兼容文件 | 资源内部页面（遵守组织模式，见 `@07-资源组织结构.md`） |
| 组织结构 | 无视，纯文件级左右切换 | 遵守（章节 / 平铺网格 / 画廊） |
| PDF | 作为同级文件，左右滑到其他图片/视频 | 本身作为资源，单页翻页 |
| 进度记忆 | 不记忆 | 不记忆 |
| 工具栏标题 | 文件名 + 序号/总数 | 资源名 + 页码/总页数 |

- 可左右滑动浏览当前目录下**同一层级**的所有兼容文件（图片/PDF/视频混合）
- 按当前排序顺序浏览
- 视频与图片在统一查看器内混合浏览（见 `@04-资源查看器.md` §4.4）
- 滑动条、缩放、加载失败阻断等交互与资源库查看器一致

#### 4.3.4 排序

| 排序方式 | 说明 |
|---------|------|
| 按文件名（默认） | 按名称字母升序排列，文件夹优先于文件 |
| 按修改时间 | 按文件/文件夹最后修改时间降序排列 |

- **独立排序**：每个目录层级独立记忆排序设置，互不影响
- 默认行为：首次进入任意目录时，统一使用"按文件名"排序

### 4.4 多选模式

#### 4.4.1 入口 & 退出

- **入口**：文件浏览器右上角 [多选] 按钮
- **退出**：点击 [取消] 或完成操作后自动退出
- 网格和列表视图均支持多选

#### 4.4.2 多选模式布局

```
┌──────────────────────────────────────────────┐
│  ← 返回   已选 N 项    [全选] [取消]          │  ← 多选模式顶部栏
├──────────────────────────────────────────────┤
│  📁 > Comics > 海贼王                        │
├──────────────────────────────────────────────┤
│  ☑ 📁 海贼王                                 │  ← 仅当前层级多选
│  ☑ 📁 进击的巨人                             │
│  □ 📁 鬼灭之刃                               │
│  □ 📄 设定集.pdf                             │
│  □ 🎬 OP_特典.mp4                            │
├──────────────────────────────────────────────┤
│  [批量添加资源]  [批量打标签]    [取消]        │  ← 底部操作栏
└──────────────────────────────────────────────┘
```

#### 4.4.3 操作说明

| 操作 | 行为 |
|------|------|
| 点击项 | 切换勾选/取消勾选 |
| 点"全选" | 勾选当前目录下全部文件夹和文件 |
| 底部"批量添加资源" | 进入批量添加流程（见 §4.5，直接添加当前层级勾选项，不经 ResourcePicker） |
| 底部"批量打标签" | 进入批量打标签流程（见 4.6） |

> **文件浏览器多选仅支持当前层级勾选**。如需递归选择深层子文件夹，使用长按/右键未入库项 → "扫描入库" → ResourcePicker 弹窗（纯树形级联选择，见 `@06-页面结构与交互.md`）。

### 4.5 批量添加资源

#### 4.5.1 完整流程

```
多选 N 项 → 点"批量添加资源"
  │
  ├─ ① 系统预过滤：
  │     - 跳过已入库项（基于 sourceId + relativePath 去重）
  │     - 跳过空文件夹
  │     - 统计：实际将添加 M 项 / 跳过 K 项
  │
  ├─ ② 确认弹窗：
  │     "将添加 8 个资源"
  │     "(已跳过 2 个重复、1 个空文件夹)"
  │     [取消] [下一步：打标签]
  │
  ├─ ③ 标签选择弹窗（可选）：
  │     ┌─────────────────────────────────────┐
  │     │  为本次添加的资源打上标签（可选）     │
  │     │  ⭐ 收藏                             │
  │     │  ☐ 热血    ☐ 已完结    ☑ 未读       │
  │     │  [+ 新建标签]                        │
  │     │                                      │
  │     │       [跳过]    [确认添加并打标签]    │
  │     └─────────────────────────────────────┘
  │
  └─ ④ 确认 → 创建 Resource + ResourceTag → 生成缩略图 → 返回刷新首页
```

#### 4.5.2 约束

- 选择"跳过"则不绑定任何标签，仅添加资源
- 已入库项自动跳过，不可重复添加
- 添加后系统自动判定组织模式（见 `@07-资源组织结构.md`）

### 4.6 批量打标签

仅在多选模式下对**已入库资源**打标签。

```
多选 N 项 → 点"批量打标签"
  │
  ├─ ① 过滤：只处理已入库项
  │     不在库的掠过
  │
  ├─ ② 提示弹窗：
  │     "将为 5 个资源打标签"
  │     "(已跳过 2 个未入库项)"
  │
  ├─ ③ 标签选择弹窗（同批量添加的标签弹窗）
  │
  └─ ④ 确认 → 批量写入 ResourceTag → 完成
```

### 4.7 删除数据源

**流程**：
- 源列表 → `···` 菜单 → "删除"
- 弹出二次确认弹窗，需用户手动确认：
  - 弹窗内容："确定要删除数据源'xxx'吗？该源下的 12 个资源将被移除，绑定的标签关联也会一并清除。标签本身会保留。"
  - 提供 [取消] 和 [确认删除] 两个按钮
- 用户手动确认后：删除 Source 记录，级联删除所有关联的 Resource 及 ResourceTag 记录
- Tag 本身保留（可能还关联其他源的资源）
- 缩略图缓存一并清理

### 4.8 编辑 SMB 凭据

当 SMB 密码过期或变更时，用户可更新凭据而无需删除重建源。

**入口**：源列表 → `···` 菜单 → "编辑 SMB 凭据"（仅 SMB 源显示此项）

**流程**：
```
··· 菜单 → "编辑 SMB 凭据"
  → 弹出凭据编辑表单（预填用户名/域，密码留空需重新输入）
  → 用户填写新密码
  → 点击"测试连接"验证新凭据
  → 测试成功 → "保存"按钮可点击
  → 保存 → 更新 flutter_secure_storage 中的密码
  → 源状态恢复为"正常"（若之前因密码失效标记为"不可用"）
```

**约束**：
- 仅 SMB 源显示"编辑 SMB 凭据"菜单项，本地源不显示
- 密码字段始终留空需重新输入（不回显旧密码，安全考虑）
- 测试连接成功后才能保存
- 凭据更新后自动尝试重新连接，源状态刷新

## 5. 边界条件

1. **SMB 连接失败**：top toast 提示具体原因，源标记为"不可用"
2. **路径不存在**（U 盘拔出、网络断开等）：源标记"不可用"，资源在首页置灰
3. **路径恢复可用**：自动或手动同步后恢复正常状态
4. **大目录浏览**：文件浏览器按需加载子目录内容（P1）
5. **空目录**：文件浏览器中显示"此文件夹为空"，添加资源时不可勾选（自动跳过），多选层叠选择时跳过
6. **无兼容文件**：文件浏览器中只显示文件夹和可识别格式文件（图片/PDF/视频/GIF），其他文件隐藏或有"不支持"标记
7. **去重**：同一 sourceId + relativePath 不可重复添加
8. **SMB 密码安全**：存储时加密，界面展示脱敏
9. **批量打标签跳过**：不在资源库的项自动跳过，弹窗提示跳过数量
