# 04 — 资源查看器

> 所属模块：D · 优先级：P0 · 状态：已定稿

---

## 1. 模块概述

资源查看器是用户消费内容的核心入口，图片和 PDF 采用统一的全屏单页左右翻页体验。可以从以下两个来源进入：

- **首页**：点击资源项进入，按资源的组织模式呈现（直接阅读 / 章节选择 / 平铺网格 / 画廊），见 [@07-资源组织结构.md](07-资源组织结构.md)
- **文件浏览器**：点击图片/PDF/GIF 等文件进入，可左右滑动浏览当前目录下所有同层级兼容文件，见 [@03-数据源管理.md](03-数据源管理.md)

查看器默认黑色背景，软件整体支持深色模式切换。

## 4. 详细需求

### 4.1 统一查看器（图片 & PDF）

图片和 PDF 采用相同的全屏单页左右翻页体验。PDF 仅渲染模式与图片不同，交互完全一致。默认黑色背景。

#### 4.1.1 页面结构

```
工具栏隐藏（沉浸）              工具栏可见
┌────────────────────┐        ┌────────────────────┐
│                    │        │  ← 返回  海贼王 (3/176) ⚙│  ← 顶部工具栏
│                    │        ├────────────────────┤
│                    │        │                    │
│                    │        │                    │
│    当前图片/PDF页   │        │    当前图片/PDF页   │
│                    │        │                    │
│                    │        │                    │
│                    │        ├────────────────────┤
│                    │        │ ●●●━━━●━━━━━━━━ 176│  ← 底部滑动条
└────────────────────┘        └────────────────────┘
```

- **顶部工具栏**（可隐藏）：返回按钮 + 资源名/文件名 + 当前页/总页数 + 设置按钮（⚙）
- **设置按钮**：翻页方向 / 双页显示开关 / 组织模式切换
- **底部滑动条**：非沉浸模式时显示，每个点代表一页，拖动滑块快速定位
- **单击**：切换工具栏 + 滑动条的显示/隐藏

#### 4.1.2 翻页交互

| 操作                    | 行为                         |
| ----------------------- | ---------------------------- |
| 向左滑动 / 点击右侧区域 | 下一页                       |
| 向右滑动 / 点击左侧区域 | 上一页                       |
| 键盘 ← / →（桌面端）  | 上一页 / 下一页              |
| 键盘 ESC（桌面端）      | 返回                         |
| 双指捏合 / 滚轮         | 缩放（0.5x ~ 5x）            |
| 拖拽（已放大时）        | 平移图片                     |
| 双击                    | 切换 1x / 适合屏幕 / 2x 缩放 |

#### 4.1.3 滑动条

- 仅在工具栏可见时显示（非沉浸模式）
- 位置：屏幕底部
- 形式：横向滑动条，总长度对应总页数，滑块每格一页
- 标签：滑动条两端标注"1"和"176"
- 拖动时实时更新当前页码和预览图（P1：缩略浮窗）
- 拖动松手后跳转到对应页
- 特殊处理：总页数仅 1 页时不显示滑动条

#### 4.1.4 跨章节连续阅读

> 仅适用于章节模式的资源（见 [@07-资源组织结构.md](07-资源组织结构.md)）。

| 场景                   | 行为                                                          |
| ---------------------- | ------------------------------------------------------------- |
| 当前在末页 + 继续左滑  | 弹出提示"下一章: 第002话"，继续滑动则进入下一章首页           |
| 当前在首页 + 继续右滑  | 弹出提示"上一章: 第001话"，继续滑动则进入上一章末页           |
| 当前已是最后一章的末页 | 弹出提示"已是最后一章"，不跳转                                |
| 提示样式               | 屏幕中央半透明文字，停留 1.5 秒后自动消失（如果用户停止滑动） |

- 进入新章节时页码重置，工具栏显示章节名 + 页码
- 工具栏设置中可关闭跨章节连续阅读

#### 4.1.5 图片加载

- 进入时预加载当前页 + 前 1 页 + 后 2 页
- 翻页时渐进加载，P1 支持先显示低质量缩略图再替换原图
- **加载失败**：显示"加载失败"占位图 + 重试按钮，**不自动跳过**，用户点击重试或手动翻到其他页

#### 4.1.6 支持格式

| 格式            | 优先级 | 说明                                 |
| --------------- | ------ | ------------------------------------ |
| JPEG, PNG       | P0     |                                      |
| GIF（播放动画） | P0     |                                      |
| WebP            | P0     |                                      |
| PDF             | P0     | 渲染为单页图片，与图片查看器统一交互 |
| BMP             | P1     |                                      |

#### 4.1.7 翻页方向

- 默认：左滑下一页（日漫模式，右→左阅读）
- 设置中可切换为"右滑下一页"（左→右阅读，如杂志）
- 切换后全局生效，记忆用户选择

#### 4.1.8 双页并排显示

| 条件                     | 行为                                            |
| ------------------------ | ----------------------------------------------- |
| 宽度 ≥ 900dp 且用户开启 | 双页并排，左页在前右页在后                      |
| 宽度 < 900dp             | 自动回退单页，禁用双页选项或自动关闭            |
| 用户手动关闭             | 始终单页，忽略宽度                              |
| 双页模式翻页             | 每次翻两页（左滑 → 显示 p3+p4）                |
| 首尾页                   | 第 1 页单独显示（右半留空或封面跨页），末页同理 |

### 4.2 两个来源的差异

查看器可从资源库首页或文件浏览器进入，两个来源的浏览上下文有本质区别：

| 维度       | 资源库首页进入                                           | 文件浏览器进入                          |
| ---------- | -------------------------------------------------------- | --------------------------------------- |
| 翻页范围   | 资源内部页面，遵守组织模式（见 [@07-资源组织结构.md](07-资源组织结构.md)） | 当前目录下同层级兼容文件，无视组织模式  |
| 组织结构   | 直接阅读 / 章节 / 平铺网格 / 画廊四种模式                | 纯文件级，左右滑动切换                  |
| PDF        | 作为资源，单页翻页                                       | 作为同级文件之一，左右滑到其他图片/视频 |
| 跨章节     | 章节模式支持跨章节连续阅读                               | 不支持（无章节概念）                    |
| 进度记忆   | 不记忆                                                   | 不记忆                                  |
| 工具栏标题 | 资源名 + 页码/总页数                                     | 文件名 + 序号/总数                      |
| 退出目标   | 返回首页（Tab 0）                                        | 返回文件浏览器                          |

### 4.3 从文件浏览器进入查看器

当用户在文件浏览器中点击图片/PDF/GIF 等兼容文件进入查看器时：

- **来源上下文**：当前文件所在目录的所有兼容文件构成"浏览列表"
- **排序**：按文件浏览器的当前排序设置
- **翻页范围**：仅浏览同目录下的兼容文件，不跨界浏览父目录或子目录
- **工具栏**：显示文件名 + 当前序号/总数
- **退出**：返回文件浏览器，恢复之前位置和状态
- 滑动条、缩放、加载失败阻断等交互与 `4.1` 一致

### 4.4 视频播放器（P2）

> 视频播放交互细节较多，后续详细讨论。当前定义关键交互。

#### 4.4.1 显示/隐藏 & 播放控制

| 操作             | 行为                                |
| ---------------- | ----------------------------------- |
| 单击视频画面     | 切换工具栏 + 进度条的显示/隐藏      |
| 双击视频画面     | 播放 / 暂停                         |
| 进度条旁暂停按钮 | 点击暂停（PC 友好，移动端同样可用） |

#### 4.4.2 进度条与文件切换分区

```
工具栏隐藏（沉浸）              工具栏可见
┌────────────────────┐        ┌────────────────────┐
│                    │        │  ← 返回  文件名.mp4  │
│                    │        ├────────────────────┤
│                    │        │                    │
│   左右滑动 = 切文件  │        │   左右滑动 = 切文件  │  ← 上部 70% 区域
│                    │        │                    │
│                    │        │                    │
├                    ┤        ├────────────────────┤
│                    │        │ ═══●═════════ 12:34 ⏸│  ← 下部 30%：进度条+暂停
└────────────────────┘        └────────────────────┘
```

- 进度条隐藏时：全屏区域左右滑动 = 切换文件
- 进度条可见时：上部 70% 左右滑动 = 切换文件；下部 30% 拖动进度条
- 仅一个视频文件时：左右滑动恢复为视频内快进快退

#### 4.4.3 长按倍速（移动端）/ 右键长按（桌面端）

| 操作                       | 行为                                                       |
| -------------------------- | ---------------------------------------------------------- |
| 长按播放区域               | 进入倍速模式，默认**2x**                             |
| 长按期间**上下滑动** | 切换倍速档位                                               |
| 松手                       | 恢复原速 1x                                                |
| 长按反馈位置               | 屏幕**中下方**（靠近手指操作区域），显示当前倍速标识 |

**倍速档位**（上下滑动切换）：

```
     上滑增速
   3.0x  ─
   2.5x  ─
   2.0x  ─  ← 长按默认
   1.75x ─
   1.5x  ─
   1.25x ─
   1.0x  ─
     下滑减速
```

- 长按不松手 + 上滑一档 = 下一个倍速位
- 长按不松手 + 下滑一档 = 上一个倍速位
- 档位吸附，不会停在中间值
- 视觉反馈：当前倍速值实时更新

> 操作模式（手势 vs 按钮布局）后续详细讨论。

## 5. 边界条件

1. **单张图片**：不显示滑动条和页数信息，只显示文件名
2. **超大分辨率图片**（>10000px 单边）：缩放加载，避免 OOM
3. **损坏的图片文件**：显示"加载失败"占位 + 重试按钮，不跳过
4. **跨章节边界（首章首页 / 末章末页）**：提示"已是第一章/最后一章"
5. **PDF 字体缺失**：使用后备字体渲染
6. **PDF 加密**：提示"此 PDF 已加密，暂不支持查看"（P2 支持密码输入）
7. **退出查看器**：清除已释放的页面缓存，首页资源不记忆阅读位置
8. **宽度临界点**：双页 ↔ 单页切换阈值 900dp，窗口 resize 时实时判断
