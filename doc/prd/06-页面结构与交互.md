## 2. 全局导航

### 2.1 移动端（手机）

使用 **BottomNavigationBar**，固定在屏幕底部，3 个 Tab：

```
┌─────────────────────────┐
│                         │
│        页面内容          │
│                         │
├────────┬────────┬───────┤
│  🏠    │  📂    │  ⚙   │
│ 首页库  │ 数据源  │ 设置  │
└────────┴────────┴───────┘
```

- 3 个 Tab，图标 + 文字
- 点击切换，保持各 Tab 的滚动位置和状态

### 2.2 平板 & 桌面端

使用 **NavigationRail**（侧边导航栏）或常驻侧栏：

```
┌────┬──────────────────────────────────┐
│ 🏠 │                                  │
│ 首 │        页面内容                   │
│ 页 │                                  │
│ 库 │                                  │
│    │                                  │
│ 📂 │                                  │
│ 数 │                                  │
│ 据 │                                  │
│ 源 │                                  │
│    │                                  │
│ ⚙  │                                  │
│ 设 │                                  │
│ 置 │                                  │
└────┴──────────────────────────────────┘
```

- 侧栏宽度：72dp（仅图标）或 256dp（图标 + 文字），可通过汉堡菜单折叠
- 平板（< 900dp）：折叠式侧栏，默认隐藏，点击汉堡图标展开
- 桌面（> 900dp）：常驻侧栏

## 4. 跨页面交互

### 4.1 首页 → 查看器 → 返回

- 进入查看器时，首页的滚动位置和标签筛选状态保持不变
- 从查看器返回时，恢复之前的滚动位置
- 如果资源在查看过程中名称/标签发生变更，返回后首页数据刷新

### 4.2 源管理 → 文件浏览器 → 添加资源 → 返回

- 添加资源后，关闭文件浏览器回到源列表页
- 源列表中的"资源数量"实时更新
- 切换到首页 Tab 时，刷新资源网格

### 4.3 标签筛选 → 查看资源 → 返回

- 已在筛选状态进入查看器，返回后筛选状态保持
- 筛选栏高亮保持

### 4.4 数据一致性

| 场景 | 同步策略 |
|------|---------|
| Tab 切换 | 每次切换到首页 Tab 时静默刷新资源列表 |
| 扫描完成 | 首页收到通知后刷新网格 |
| 标签变更 | 关联的资源在首页名字/标签信息更新 |

### 4.5 ResourcePicker 弹窗组件

ResourcePicker 是一个**可复用的弹窗组件**，用于扫描文件夹结构并让用户以树形选择需要添加/拆分的子项。宽度 < 900dp 和 ≥ 900dp 使用同一布局，仅弹窗尺寸不同。

#### 4.5.1 布局

```
┌──────────────────────────────────────┐
│ 关闭    扫描入库：全部漫画  已选 3 项 │  ← 顶部栏
├──────────────────────────────────────┤
│ 📁 全部漫画           [全选子项] →    │  ← 仅导航，无复选框；[全选子项] 可一键勾选直接子项
│  ☑ 📁 海贼王                        │  ← 勾选 → 海贼王作为一个资源
│  ☐ 📁 进击的巨人                     │
│  ☐ 📁 鬼灭之刃                      │
│  ☑ 📄 设定集.pdf                    │
│  ☑ 📁 画集                          │
├──────────────────────────────────────┤
│  已选 3 项                           │
│  [批量添加资源]  [批量打标签]  [取消]  │  ← 底部按钮随场景变化
└──────────────────────────────────────┘
```

- 纯树形结构，无右侧内容区
- 窄屏与宽屏一致，仅弹窗宽度不同

#### 4.5.2 树形交互

**核心原则**：勾选 = 该节点作为一个独立资源，不级联不递归。

| 操作 | 行为 |
|------|------|
| 点节点复选框 | 该节点本身勾选/取消（独立，不影响父子） |
| 点击节点名 / 展开箭头 | 展开/折叠子树，纯导航，不改变勾选 |
| 点 [全选子项] | 一键勾选该节点的所有直接子项（仅一层，不递归） |
| 空文件夹 | 不显示 |

**[全选子项] 按钮出现条件**：

| 条件 | 是否显示 |
|------|---------|
| 该节点下有子文件夹（可作为资源导入） | 显示 |
| 该节点下仅含图片/PDF 等文件（无需拆分） | 不显示 |
| 该节点下无可导入的子项 | 不显示 |

**复选框出现条件**：
- 仅显示在可直接作为资源单元导入的节点上
- 如果某文件夹下一层全是图片（单一资源），该文件夹在树中显示为一个带复选框的叶子节点，不再展开
- 如果某文件夹下有子文件夹 + 文件混合，则展开显示，每个子文件夹和文件各自带复选框

**智能识别规则**：

| 目录结构 | 树表现 |
|---------|--------|
| /海贼王/001.jpg, 002.jpg | 一个节点，带复选框，不可展开（= 直接阅读型资源） |
| /海贼王/卷01/001.jpg, 卷02/001.jpg | 一层节点带复选框 + [全选子项] |
| /混合/子文件夹A/, 设定.pdf | 节点展开，子文件夹和 PDF 各自带复选框 |

#### 4.5.3 边界条件

1. 子文件夹全是图片：不显示 [全选子项]，该节点自身带复选框即可
2. 空文件夹：树中不显示
3. 文件类型过滤：仅展示文件夹 + 可识别的兼容文件（图片/PDF/压缩包/视频），不兼容格式不展示
4. 根节点复选框：无——根节点仅导航 + [全选子项]
5. 去重：重复项不显示
6. 大量子节点（>100）：P2 虚拟滚动，当前不做特殊优化

#### 4.5.4 调用入口

| 场景 | 入口 | 弹窗标题 | 底部按钮 |
|------|------|---------|---------|
| 文件浏览器长按→"扫描入库" | [@03-数据源管理.md](03-数据源管理.md) | 扫描入库：xxx | `[批量添加资源] [批量打标签]` |
| 文件浏览器多选→批量添加 | [@03-数据源管理.md](03-数据源管理.md) | 扫描入库 | `[批量添加资源] [批量打标签]` |
| 首页长按→"拆分资源" | [@01-资源库首页.md](01-资源库首页.md) | 拆分资源：xxx | `[拆分并保留原资源] [拆分并删除原资源]` |

#### 4.5.5 后续操作

添加/拆分确认后，可选弹出标签选择弹窗（复用 [@02-标签系统.md](02-标签系统.md) 组件），标签**完全重新选择**，不继承原资源标签。

## 5. 平台适配差异

| 交互 | 手机 | 平板 | 桌面 |
|------|------|------|------|
| 主导航 | BottomNavigationBar | NavigationRail（可折叠） | NavigationRail（常驻） |
| 屏幕方向 | 仅查看器横屏，其余页强制竖屏 | 全响应式 | 全响应式 |
| 多选触发 | 长按 | 长按 | 右键 |
| 右键菜单 | 无 | 部分支持 | 全面支持（含右键长按倍速） |
| 键盘快捷键 | 无 | 蓝牙键盘支持 | 全面支持 |
| 返回按钮 | 系统返回键 / 手势 | 顶部返回箭头 | 顶部返回箭头 + ESC |
| 手势返回 | 不支持 | 不支持 | 不支持 |
| 查看器翻页 | 左右滑动 | 左右滑动 / 方向键 | 方向键 / 滚轮 |
| 查看器双页显示 | 不可用（宽度不足） | 横屏 ≥900dp 可用 | ≥900dp 可用，用户可选 |
| 暗黑模式 | UI 主题色可变，查看器始终黑底 | 同手机 | 同手机 |
| 视频倍速 | 长按 + 上下滑动 | 长按 + 上下滑动 | 右键长按 + 鼠标上下 |
| 缩略图列数 | 3 列 | 4-5 列 | 5-6 列 |
| 文件选择器 | 系统原生 | 系统原生 | 系统原生 |
| 缩放手势 | 双指捏合 | 双指捏合 / 触控板 | 滚轮 + Ctrl |
