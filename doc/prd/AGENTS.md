## 产品定位

Resource Viewer 是一款多数据源统一收藏管理工具，帮助用户集中浏览、检索分散存储的漫画、图包等资源。

核心能力：竖版缩略图网格浏览 → 内置收藏 + 自定义标签交集筛选 → 统一查看器翻阅图片/PDF。

---

## 术语定义

| 术语 | 定义 |
|------|------|
| **数据源（Source）** | 用户添加的文件来源，如本地文件夹路径或 SMB 网络共享 |
| **资源（Resource）** | 一个可浏览/阅读的内容单元，通常为一个文件夹或单个 PDF/压缩包文件 |
| **标签（Tag）** | 分为内置标签（收藏）和自定义标签，可绑定到任意资源 |
| **组织模式** | 资源内部文件的组织方式：章节模式 / 平铺网格 / 画廊模式 |
| **ResourcePicker** | 可复用的树形扫描选择弹窗组件，用于深层子文件夹选择 |

## 核心流程

```
打开应用
  │
  ├─ [首次使用] → 添加数据源（本地 / SMB）→ 文件浏览器手动勾选 → 添加资源
  │     └─ 系统自动判定组织模式（章节 / 平铺网格 / 画廊）
  │
  └─ [日常使用]
      ├─ Tab 0 首页：浏览缩略图网格 → 筛选栏（全部 | 收藏 | 自定义标签交集）
      ├─ Tab 1 数据源：管理源 → 文件浏览器（导航/多选/扫描入库）
      │     └─ 长按文件夹 → 扫描入库 → ResourcePicker 弹窗（树形级联选择）
      └─ Tab 2 设置：缓存清理、外观切换、查看器默认配置
```
