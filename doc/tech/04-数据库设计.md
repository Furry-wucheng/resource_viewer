# 04 — 数据库设计

> 日期：2026-06-19 · 状态：已定稿 · 依据：[@prd/05-数据模型.md](../prd/05-数据模型.md) · [@02-架构设计.md](02-架构设计.md) · [@03-性能优化设计.md](03-性能优化设计.md) §5

---

## 1. 概述

采用 **drift**（SQLite ORM）实现，四个表对应 [@prd/05-数据模型.md](../prd/05-数据模型.md) 的完整实体定义。数据库运行在后台 Isolate（`NativeDatabase.createInBackground`），启用 WAL 模式 + foreign_keys。

---

## 2. 实体关系

```
Source (1) ──→ (N) Resource (N) ──→ (M) Tag
                    │                       │
                    └── ResourceTag ───────┘  (junction)
```

---

## 3. 枚举定义

```dart
enum SourceType { local, smb, ftp, webdav }
enum ResourceType { folder, pdf, archive, video }
enum OrganizationMode { chapter, flatgrid, gallery }
```

`ftp` 和 `webdav` 为架构预留（[@02-架构设计.md §4.1](02-架构设计.md)），当前仅使用 `local` 和 `smb`。

## 4. 表定义

### 4.1 `sources` — 数据源

> 对应 [@prd/05](../prd/05-数据模型.md) §2.1

| 列 | Drift 类型 | 约束 | 说明 |
|----|-----------|------|------|
| `id` | `text()` | PK | UUIDv4 |
| `name` | `text().withLength(max: 100)()` | NOT NULL | 用户自定义名称 |
| `type` | `textEnum<SourceType>()` | NOT NULL | `local` / `smb` / `ftp` / `webdav` |
| `rootPath` | `text().withLength(max: 500)()` | NOT NULL | `D:\Comics` 或 `smb://192.168.1.100/share` |
| `host` | `text().withLength(max: 255).nullable()` | | 网络源主机地址 |
| `port` | `integer().nullable()` | | 默认 445 |
| `username` | `text().withLength(max: 100).nullable()` | | |
| `passwordStored` | `boolean().withDefault(const Constant(false))()` | NOT NULL | 密码存 `flutter_secure_storage`，此列不存密码 |
| `domain` | `text().withLength(max: 100).nullable()` | | 域/工作组 |
| `enabled` | `boolean().withDefault(const Constant(true))()` | NOT NULL | 关闭后该源资源不在首页显示 |
| `isAvailable` | `boolean().withDefault(const Constant(false))()` | NOT NULL | 当前是否可达 |
| `lastCheckAt` | `dateTime().nullable()` | | 上次连接检查时间 |
| `createdAt` | `dateTime().withDefault(currentDateAndTime)()` | NOT NULL | |
| `updatedAt` | `dateTime().withDefault(currentDateAndTime)()` | NOT NULL | |

**索引**：`type` 无额外索引（值分布少，全表扫描可接受）。

**密码安全**（[@02 §8](02-架构设计.md)）：

- `passwordStored` = `true` → 可通过 `FlutterSecureStorage.read(key: 'smb_pwd_{id}')` 取出实际密码
- `passwordStored` = `false` → 匿名访问或未配置密码
- 界面展示脱敏：`isPasswordStored ? '****' : ''`

### 4.2 `resources` — 资源

> 对应 [@prd/05](../prd/05-数据模型.md) §2.2

| 列 | Drift 类型 | 约束 | 说明 |
|----|-----------|------|------|
| `id` | `text()` | PK | UUIDv4 |
| `sourceId` | `text().references(Sources, #id, onDelete: KeyAction.cascade)()` | FK, NOT NULL | 所属数据源 |
| `name` | `text().withLength(max: 255)()` | NOT NULL | 资源名称（文件夹名或文件名） |
| `type` | `textEnum<ResourceType>()` | NOT NULL | `folder` / `pdf` / `archive` / `video` |
| `organizationMode` | `textEnum<OrganizationMode>().nullable()` | | `null`=未判定，判定后为 `chapter`/`flatgrid`/`gallery` |
| `relativePath` | `text().withLength(max: 1000)()` | NOT NULL | 相对于源根目录的路径 |
| `thumbnailPath` | `text().withLength(max: 500).nullable()` | | 本地缓存缩略图路径 |
| `fileCount` | `integer().nullable()` | | 内部图片/页数 |
| `fileSize` | `int64().nullable()` | | 字节 |
| `isAvailable` | `boolean().withDefault(const Constant(true))()` | NOT NULL | 文件当前是否可访问 |
| `lastScannedAt` | `dateTime().nullable()` | | 最后扫描时间 |
| `createdAt` | `dateTime().withDefault(currentDateAndTime)()` | NOT NULL | |
| `updatedAt` | `dateTime().withDefault(currentDateAndTime)()` | NOT NULL | |

**Unique**：`(sourceId, relativePath)` — 同一来源下同一路径不可重复添加；不同来源可各自添加同名路径。

**索引**：

| 索引 | 类型 | 目的 |
|------|------|------|
| `(sourceId, relativePath)` | UNIQUE | 扫描去重 + upsert 冲突检测 |
| `(sourceId)` | FK 自动建立 | 级联删除 + 按源查询资源 |
| `(createdAt, id)` | 复合索引 | 按时间排序 + 键集分页（[@03 §5.3](03-性能优化设计.md)） |
| `(name, id)` | 复合索引 | 按名称排序 + 键集分页 |

### 4.3 `tags` — 标签

> 对应 [@prd/05](../prd/05-数据模型.md) §2.3 · [@prd/02-标签系统.md](../prd/02-标签系统.md)

| 列 | Drift 类型 | 约束 | 说明 |
|----|-----------|------|------|
| `id` | `text()` | PK | UUIDv4 |
| `name` | `text().withLength(max: 20)()` | NOT NULL, UNIQUE | 标签名称 |
| `color` | `text().withLength(max: 7)()` | NOT NULL | HEX 颜色值 `#RRGGBB` |
| `isBuiltIn` | `boolean().withDefault(const Constant(false))()` | NOT NULL | 内置标签不可删除/重命名 |
| `createdAt` | `dateTime().withDefault(currentDateAndTime)()` | NOT NULL | |
| `updatedAt` | `dateTime().withDefault(currentDateAndTime)()` | NOT NULL | |

**索引**：`name` UNIQUE 自动建立。内置标签标准 `isBuiltIn = true` 索引无需单独建（值分布极端）。

### 4.4 `resource_tags` — 资源-标签关联

> 对应 [@prd/05](../prd/05-数据模型.md) §2.4

| 列 | Drift 类型 | 约束 | 说明 |
|----|-----------|------|------|
| `resourceId` | `text().references(Resources, #id, onDelete: KeyAction.cascade)()` | PK 之一 | |
| `tagId` | `text().references(Tags, #id, onDelete: KeyAction.cascade)()` | PK 之一 | |
| `createdAt` | `dateTime().withDefault(currentDateAndTime)()` | NOT NULL | |

**PK**：`(resourceId, tagId)` — 联合唯一约束，自动建 `(resource_id, tag_id)` 索引。

**索引**：

| 索引 | 目的 |
|------|------|
| `(resource_id, tag_id)` | PK 自动建立 — 查"某资源的标签" |
| `(tag_id, resource_id)` | **手动添加**（`@TableIndex.sql`） — 查"某标签的资源"，标签交集筛选核心路径（[@03 §5.1](03-性能优化设计.md)） |

## 5. 级联删除

> 来源 [@prd/05](../prd/05-数据模型.md) §5

均通过 drift `onDelete: KeyAction.cascade` + `PRAGMA foreign_keys = ON` 保证：

| 操作 | Resource | Tag | ResourceTag |
|------|----------|-----|-------------|
| 删除 Source | 级联删除 | 保留 | 级联删除 |
| 删除 Resource | 删除 | 保留 | 级联删除 |
| 删除 Tag | 保留 | 删除 | 级联删除 |
| 资源不可达 | `isAvailable = false` | 不影响 | 不影响 |

> `PRAGMA foreign_keys = ON` 是**每连接必须执行**的 pragma（SQLite 默认关闭）。通过 `beforeOpen` 确保。

## 6. 内置标签播种

> 对应 [@prd/02](../prd/02-标签系统.md) §4.1 "首次启动自动创建，不存在则补建"

**首次安装**（`MigrationStrategy.onCreate`）：

```dart
onCreate: (m) async {
  await m.createAll();
  await into(tags).insert(TagsCompanion.insert(
    id: '00000000-0000-0000-0000-000000000001',
    name: '收藏',
    color: '#FFC107',
    isBuiltIn: const Value(true),
  ));
},
```

**启动补建**（统一放 `beforeOpen`，对首次安装与升级均生效，idempotent）：

```dart
beforeOpen: (m) async {
  // 补建内置标签（首次安装 onCreate 已播种，此处对升级用户兜底）
  final count = await (select(tags)..where((t) => t.isBuiltIn.equals(true))).get();
  if (count.isEmpty) {
    await into(tags).insert(TagsCompanion.insert(
      id: '00000000-0000-0000-0000-000000000001',
      name: '收藏',
      color: '#FFC107',
      isBuiltIn: const Value(true),
    ));
  }
},
```

> **UUID 约定例外**：§3 规定实体 ID 使用 UUIDv4，但内置标签使用固定 UUID `00000000-0000-0000-0000-000000000001`（非 v4 格式）以便跨版本识别和补建去重。其他实体（Source/Resource/自定义 Tag）均用 `uuid` 包生成的 v4。

## 7. 数据库初始化

```dart
// lib/data/services/database_service.dart
QueryExecutor _openConnection() {
  return NativeDatabase.createInBackground(
    _dbFile(),
    setup: (db) {
      db.execute('PRAGMA journal_mode = WAL;');
      db.execute('PRAGMA foreign_keys = ON;');
      db.execute('PRAGMA synchronous = NORMAL;');
    },
    readPool: 3,
  );
}
```

| PRAGMA | 作用 |
|--------|------|
| `journal_mode = WAL` | 并发读 + 单写分离；`readPool` 必需 |
| `foreign_keys = ON` | 级联删除生效（SQLite 默认关闭，每连接须重设） |
| `synchronous = NORMAL` | WAL 下安全，比 FULL 快 |
| `readPool: 3` | 多 Isolate 并行读，大扫描不阻塞小查询 |

> 来自 [@03-性能优化设计.md §5.6](03-性能优化设计.md)

## 8. 关键 DAO 查询

### 8.1 标签交集筛选

```dart
Selectable<Resource> filterByTags(List<String> tagIds) {
  if (tagIds.isEmpty) return select(resources); // 空标签返回全部（首页默认状态）

  // 防御性去重，避免重复 tagId 导致 COUNT(*) 永不等于 length
  final ids = tagIds.toSet().toList();

  final matchedIds = selectOnly(resourceTags)
    ..addColumns([resourceTags.resourceId])
    ..where(resourceTags.tagId.isIn(ids))
    ..groupBy([resourceTags.resourceId])
    ..having(countAll().equals(ids.length));

  return select(resources)
    ..where((r) => r.id.isInQuery(matchedIds));
}
```

> 用 `GROUP BY HAVING COUNT(*) = N` 替代 INTERSECT，原因见 [@03 §5.2](03-性能优化设计.md)

### 8.2 键集分页

```dart
Future<List<Resource>> pageAfter({
  required String? lastCreatedAt,
  required String? lastId,
  required int pageSize,
}) {
  final q = select(resources)
    ..orderBy([(t) => OrderingTerm.desc(t.createdAt), (t) => OrderingTerm.asc(t.id)])
    ..limit(pageSize);

  if (lastCreatedAt != null && lastId != null) {
    final cursorDt = DateTime.parse(lastCreatedAt!);
    q.where((r) =>
      r.createdAt.isSmallerThanValue(cursorDt) |
      (r.createdAt.equals(cursorDt) & r.id.isBiggerThanValue(lastId!)));
  }
  return q.get();
}
```

> 需要 `(createdAt, id)` 复合索引。

### 8.3 扫描去重 / Upsert

```dart
// 冲突列 = (sourceId, relativePath)，冲突时更新元数据
await into(resources).insertOnConflictUpdate(
  row,
  target: [resources.sourceId, resources.relativePath],
);
```

### 8.4 统计查询

```dart
// 标签管理页：每个标签关联资源数
Future<Map<String, int>> tagResourceCounts() async {
  final rows = await (selectOnly(resourceTags)
    ..addColumns([resourceTags.tagId, countAll()])
    ..groupBy([resourceTags.tagId])).get();
  return {for (final r in rows) r.read(resourceTags.tagId)!: r.read(countAll())!};
}

  final matchedIds = /* same as §8.1 subquery */;
```

### 8.5 源启停过滤

> **注意**：首页资源列表不应全量 `watch()`（10000+ 资源场景见 [@03 §5.4](03-性能优化设计.md)）。此处提供 `Future` 键集分页查询；源启停状态变化用轻量 `watchSingle` 计数监听，触发首页重新 `get()`。

```dart
// 首页：仅展示 enabled + isAvailable 的源下的资源（键集分页）
Future<List<Resource>> getAvailableResources({
  required String? lastCreatedAt,
  required String? lastId,
  required int pageSize,
}) {
  final query = select(resources).join([
    innerJoin(sources, sources.id.equalsExp(resources.sourceId),
        useColumns: false),
  ])
    ..where(sources.enabled.equals(true) & resources.isAvailable.equals(true))
    ..orderBy([
      (t) => OrderingTerm.desc(t.read(resources.createdAt)),
      (t) => OrderingTerm.asc(t.read(resources.id)),
    ])
    ..limit(pageSize);
  if (lastCreatedAt != null && lastId != null) {
    query
      ..where(resources.createdAt.isSmallerThanValue(lastCreatedAt) |
          (resources.createdAt.equals(lastCreatedAt) &
              resources.id.isBiggerThanValue(lastId)));
  }
  return query.map((row) => row.readTable(resources)).get();
}
```

## 9. 迁移策略

| 版本 | 变更 | 迁移方式 |
|------|------|---------|
| 1 | 初始建表 + 播种内置标签 | `onCreate` 执行 `createAll` + 插入收藏标签 |
| 2+ | 新增列/索引/表 | `onUpgrade` 中按版本号追加 `m.addColumn` / `customStatement` |

**v2 迁移示例**（新增 `Resources.lastViewedAt` 列 + 对应索引）：

```dart
@override
MigrationStrategy get migration => MigrationStrategy(
      beforeOpen: (details) async {
        await customStatement('PRAGMA foreign_keys = ON;');
        await _ensureBuiltInTags(); // 补建内置标签，idempotent
      },
      onCreate: (m) async {
        await m.createAll();
        await _seedBuiltInTags();
      },
      onUpgrade: (m, from, to) async {
        if (from < 2) {
          // v2: 新增 lastViewedAt 列 + 索引
          await m.addColumn(resources, resources.lastViewedAt);
          await customStatement(
            'CREATE INDEX resources_last_viewed ON resources (last_viewed_at, id)',
          );
        }
        // enum 列向后兼容：新增 OrganizationMode 值（如 v3 加 'mixed'）
        // textEnum 存 TEXT，旧数据不受影响，新值读取正常
      },
    );
```

## 11. 注意项

1. **`foreign_keys` 每连接重设**：SQLite 默认关闭，`beforeOpen` 中必须重新 `PRAGMA foreign_keys = ON`
2. **`resource_tags` 手动索引**：drift `@TableIndex(columns: {#tagId, #resourceId})` 列顺序不保证，改用 `@TableIndex.sql('CREATE INDEX idx_rt_tag_resource ON resource_tags (tag_id, resource_id)')`
3. **UUID 为 TEXT**：键集分页时不能仅用 UUID 排序（随机），需配合时间或 name 作为主排序列 + UUID 为 tiebreaker
4. **密码不入库**：`passwordStored` 仅标记，实际密码在 `flutter_secure_storage` 中存取
5. **`organizationMode` = null 的含义**：由 `DetectOrganizationModeUseCase` 异步判定（[@prd/07](../prd/07-资源组织结构.md) §3.1），判定前为 null，判定后写入具体模式
