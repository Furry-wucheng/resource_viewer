# 06 — 测试策略

> 日期：2026-06-19 · 状态：已定稿 · 依据：[@02-架构设计.md](02-架构设计.md) · [@04-数据库设计.md](04-数据库设计.md) · [@05-错误处理策略.md](05-错误处理策略.md)

## 1. 概述

定义从 Domain → Repository → ViewModel → Widget → Integration 的五层测试金字塔，统一 mock 策略、目录结构和命名约定。

核心原则：**drift 用真实内存库（不 mock），外部边界用 mocktail 接口 mock。**

---

## 2. 测试金字塔

```
        ┌──────────┐
        │    E2E   │  极少数：关键用户流程（添加源 → 扫描 → 查看）
        ├──────────┤
       ┌┤  Widget  ├┐  少数：核心 UI 组件交互验证
       │└──────────┘│
      ┌┤ ViewModel ├┐  中等：状态转换 + 错误路径 + 筛选逻辑
      │└───────────┘│
     ┌┤  Repo / UC  ├┐  较多：Repository CRUD + UseCase 编排
     │└─────────────┘│
    ┌┤    Domain     ├┐  最广泛：freezed 序列化、枚举、工具函数
    └┴───────────────┘┘
```

## 3. Mock 策略

| 被替换组件 | 方案 | 工具 |
|-----------|------|------|
| drift 数据库 | **真实内存库** | `NativeDatabase.memory()` |
| `FileSource` 接口 | mock 接口 | `mocktail` |
| 各类 Service（SmbFileService、PdfRenderService、ArchiveService） | mock 接口 | `mocktail` |
| `flutter_secure_storage` | mock 接口 | `mocktail` |
| Repository（在 ViewModel / UseCase 测试中） | mock 接口 | `mocktail` |
| ViewModel（在 Widget 测试中） | mock 接口 | `mocktail` |

### 3.1 选择 `mocktail`（不选 `mockito`）

- 零代码生成，无 `build_runner` 依赖
- Dart 3 `sealed class` 兼容
- API 简洁：`when(() => mock.foo()).thenReturn(...)` / `verify(() => mock.foo()).called(1)`

### 3.2 drift 为什么用真实内存库

- SQLite 内存库极快、确定性强
- 不引入 mock 数据库的维护成本（mock SQL 行为极易与实际偏差）
- `NativeDatabase.memory()` 创建即用、隔离、并行无冲突
- 可在 `setUp` 中建表 + 播种，`tearDown` 中关闭

### 4.2 Repository 测试

| 项目 | 说明 |
|------|------|
| 对象 | SourceRepository、ResourceRepository、TagRepository、FilesystemRepository |
| 方式 | 真实 drift in-memory + mock 外部 Service |
| 覆盖 | CRUD、级联删除、唯一约束、标签交集查询、upsert 冲突 |

```dart
// test/data/repositories/tag_repository_test.dart
void main() {
  late AppDatabase db;
  late TagRepository repo;

  setUp(() async {
    db = AppDatabase(NativeDatabase.memory(
      setup: (rawDb) => rawDb.execute('PRAGMA foreign_keys = ON;'),
    ));
    // 触发 onCreate（建表 + 内置标签播种），drift 不会在构造时自动建表
    await db.customSelect('SELECT 1').get();
    repo = TagRepository(db);
  });

  tearDown(() => db.close());

  group('标签交集筛选', () {
    test('无标签时应返回全部资源', () async {
      await _createTestResource(db, name: '资源A');
      await _createTestResource(db, name: '资源B');
      final result = await repo.filterByTags([]);
      expect(result.length, 2);
    });

    test('单标签筛选应返回关联资源', () async {
      final tagId = await repo.createTag('热血', '#FF0000');
      final resourceId = await _createTestResource(db);
      await repo.addResourceTag(resourceId, tagId);

      final result = await repo.filterByTags([tagId]);
      expect(result.length, 1);
      expect(result.first.id, resourceId);
    });

    test('双标签交集：仅返回同时拥有的资源', () async {
      final tagA = await repo.createTag('热血', '#FF0000');
      final tagB = await repo.createTag('已完结', '#0000FF');
      final res1 = await _createTestResource(db, name: '海贼王');
      final res2 = await _createTestResource(db, name: '火影');

      await repo.addResourceTag(res1, tagA);
      await repo.addResourceTag(res1, tagB);
      await repo.addResourceTag(res2, tagA);

      final result = await repo.filterByTags([tagA, tagB]);
      expect(result.length, 1);
      expect(result.first.id, res1);
    });
  });

  group('级联删除', () {
    test('删除源应级联删除资源和 ResourceTag', () async {
      final sourceId = await repo.addSource(testSource);
      final resourceId = await _createTestResource(db, sourceId: sourceId);
      final tagId = await repo.createTag('标签', '#FF0000');
      await repo.addResourceTag(resourceId, tagId);

      await repo.deleteSource(sourceId);

      expect(await repo.getResource(resourceId), isNull);
      final tagCount = await repo.getTagCount(tagId);
      expect(tagCount, 0);
    });

    test('删除标签应保留资源，仅删除 ResourceTag', () async {
      final resourceId = await _createTestResource(db);
      final tagId = await repo.createTag('标签', '#FF0000');
      await repo.addResourceTag(resourceId, tagId);

      await repo.deleteTag(tagId);

      expect(await repo.getResource(resourceId), isNotNull);
      final tags = await repo.getResourceTags(resourceId);
      expect(tags, isEmpty);
    });
  });

  group('去重', () {
    test('同 sourceId + relativePath 不可重复添加', () async {
      await repo.addResource(testResource);
      expect(
        () => repo.addResource(testResource.copyWith(name: '不同名')),
        throwsA(isA<SqliteException>()),
      );
    });
  });
}
```

### 4.4 ViewModel 测试

| 项目 | 说明 |
|------|------|
| 对象 | HomeViewModel、FileBrowserViewModel、ViewerViewModel、TagViewModel、SettingsViewModel |
| 方式 | mock Repository/UseCase，监听 `notifyListeners` |
| 覆盖 | 状态转换（loading → success / error）、筛选逻辑、重试、空状态 |

```dart
// test/ui/view_models/home_view_model_test.dart
void main() {
  late MockResourceRepository mockRepo;
  late MockFilterResourcesByTagsUseCase mockFilter;
  late HomeViewModel vm;

  setUp(() {
    mockRepo = MockResourceRepository();
    mockFilter = MockFilterResourcesByTagsUseCase();
    vm = HomeViewModel(repo: mockRepo, filterUseCase: mockFilter);
  });

  group('加载资源', () {
    test('应发出 loading → success 状态', () async {
      when(() => mockRepo.getAllResources()).thenAnswer(
        (_) async => Ok([testResource]),
      );

      final states = <UiState>[];
      vm.addListener(() => states.add(vm.state));

      await vm.loadResources();

      expect(states, [UiState.loading, UiState.success]);
      expect(vm.resources, [testResource]);
      expect(vm.state, UiState.success);
    });

    test('SMB 失败应发出 loading → error + 可重试', () async {
      when(() => mockRepo.getAllResources()).thenAnswer(
        (_) async => Err(SourceUnreachableError('SMB 不可达')),
      );

      await vm.loadResources();

      expect(vm.state, UiState.error);
      expect(vm.canRetry, isTrue);
      expect(vm.errorMessage, contains('不可达'));
    });

    test('加载失败后重试应重新请求', () async {
      when(() => mockRepo.getAllResources())
        ..thenAnswer((_) async => Err(SourceUnreachableError('...')))
        ..thenAnswer((_) async => Ok([testResource]));

      await vm.loadResources();
      expect(vm.state, UiState.error);

      await vm.retry();
      expect(vm.state, UiState.success);
      verify(() => mockRepo.getAllResources()).called(2);
    });
  });

  group('标签筛选', () {
    test('选中标签应更新 selectedTagIds 并重新筛选', () async {
      when(() => mockFilter.execute(['tag1'])).thenAnswer(
        (_) async => Ok([testResource]),
      );

      await vm.selectTag('tag1');

      expect(vm.selectedTagIds, {'tag1'});
      expect(vm.resources, [testResource]);
    });

    test('选中全部应清除筛选并恢复全量', () async {
      vm.toggleSelectAll();

      expect(vm.selectedTagIds, isEmpty);
    });
  });
}
```
